# Projeto de Previsão de Compras de E-commerce

Projeto desenvolvido originalmente em 09/2025 para o módulo 37 do curso de ciência de dados da EBAC. Através de uma base de dados contendo informações e características de clientes de um e-commerce, o objetivo do projeto é a criação de modelos de aprendizado de máquina para prever se um cliente irá ou não realizar uma compra, buscando entender os padrões que diferenciam essas duas possibilidades e os distinguem, para possível tomada de decisões estratégicas. 

O código é composto pelas diversas etapas de um projeto de ciência de dados: limpeza, tratamento e análise exploratória dos dados, pré-processamento das informações, modelagem e avaliação dos algoritmos de aprendizado de máquina. Foi desenvolvido utilizando a linguagem Python e suas principais bibliotecas para a área de dados: Pandas, Matplotlib, Seaborn, Plotly e Scikit-Learn, no Jupyter Notebook. Também conta com comentários e explicações sobre cada processo feito, bem como gráficos para melhor visualização dos dados, resultando num storytelling mais fluido e prático. 

Para funcionamento eficaz do código, é necessário ter instalado todos os pacotes e bibliotecas presentes no arquivo 'requirements.txt'.

O aluno e cientista de dados Guilherme Zorzetto de Sá foi o principal colaborador desse projeto, proposto e feito em parceria com a EBAC e seus tutores.
